# Popup UI 스펙

메뉴바 아이콘 클릭 시 나타나는 미니멀 입력 팝업.
빠른 질의응답 전용. 상세 대화는 앱 창으로 이동.

## 위치

**화면 하단 중앙** 고정. 메뉴바 위치와 무관.

```
┌─────────────────── 전체 화면 ───────────────────┐
│                                                 │
│                                                 │
│                                                 │
│                                                 │
│          ┌──────────────────────────┐           │
│          │        팝업 창           │           │
│          └──────────────────────────┘           │
│              ↑ 하단에서 80px 위                  │
└─────────────────────────────────────────────────┘
```

Rust에서 트레이 클릭 시 모니터 크기 기준으로 좌표 계산:
- `x = (monitor.width - window.width) / 2`
- `y = monitor.height - window.height - 80px(logical) * scale_factor`

## 상세 구조

```
┌─ InputBar (전체 컨테이너) ──────────────────────────┐
│                                                     │
│  [TextArea]                                         │
│   placeholder: "답글..."                            │
│   Enter: 전송 / Shift+Enter: 줄바꿈                 │
│   자동 높이 확장 (최대 5줄)                          │
│                                                     │
│  ┌─ BottomRow ────────────────────────────────────┐ │
│  │  [+버튼]    [AssistantPicker ∨]    [🎙 버튼]   │ │
│  └───────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

## AssistantPicker 드롭다운

```
┌─────────────────────────────────────────────────────┐
│  비서 이름 ∨                                         │  ← 닫힌 상태
└─────────────────────────────────────────────────────┘

클릭 시 위로 펼쳐짐 ↑

┌──────────────────────────┐
│  ● 비서 A   일정 관리    │
│  ● 비서 B   링크 정리    │
│  ● 비서 C   범용 대화    │
└──────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  비서 A ∨                                           │  ← 선택된 상태
└─────────────────────────────────────────────────────┘
```

## 스타일

| 항목 | 값 |
|------|----|
| 창 크기 | 350 × auto (내용에 따라 최소 80px ~ 최대 200px) |
| 배경 | `#1c1c1e` (다크), 둥근 모서리 `12px` |
| 텍스트 | `#ebebf5` |
| placeholder | `#636366` |
| 하단 버튼 행 | `#2c2c2e` 배경 구분선 위 |
| 폰트 | `-apple-system` |

---

## 버튼 역할

| 버튼 | Phase 2 (껍데기) | Phase 4+ (실제) |
|------|------------------|-----------------|
| `+` | 자리만 | 파일 첨부 또는 앱 창 열기 |
| `비서 이름 ∨` | 드롭다운 UI만 | 비서 전환 |
| `🎙` | 자리만 | 음성 입력 |
| 텍스트 입력 + Enter | UI만 | Gemini 호출 |
