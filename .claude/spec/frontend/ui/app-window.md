# App Window UI 스펙

Dock 또는 팝업에서 진입하는 몰입형 채팅 창.
비서가 버튜버처럼 인격을 가진 캐릭터로 연출.

---

## 전체 레이아웃

```
┌─────────────────┬────────────────────────────────────────┐
│                 │                                        │
│  [에이전트 A] ← 선택됨                                   │
│  [에이전트 B]   │  ╭──────────────────────────────────╮  │
│  [에이전트 C]   │  │ 비서: 안녕하세요! 뭐 도와드릴까요? │  │
│                 │  ╰──────────────────────────────────╯  │
│                 │                                        │
│                 │          ╭──────────────────────────╮  │
│                 │          │ 나: 오늘 일정 알려줘      │  │
│                 │          ╰──────────────────────────╯  │
│                 │                                        │
│                 │  ╭──────────────────────╮              │
│  ┌───────────┐  │  │ 비서: 오늘은 3건의   │              │
│  │           │  │  │ 미리알림이 있어요.   │              │
│  │  캐릭터   │  │  ╰──────────────────────╯              │
│  │  이미지   │  │                                        │
│  │           │  ├────────────────────────────────────────┤
│  └───────────┘  │  답글...              비서 A ∨   🎙    │
└─────────────────┴────────────────────────────────────────┘

  ← 160px →  ← ──────────────── 560px ──────────────── →
                          전체: 720 × 580px
```

## 사이드바 구조

에이전트 목록과 캐릭터가 같은 구획(사이드바)에 위치.

```
┌─────────────────┐
│  에이전트 목록   │
│                 │
│  ┌───────────┐  │  ← 선택된 에이전트 하이라이트
│  │  비서 A   │  │
│  └───────────┘  │
│  ┌───────────┐  │
│  │  비서 B   │  │
│  └───────────┘  │
│  ┌───────────┐  │
│  │  비서 C   │  │
│  └───────────┘  │
│                 │
│  ─────────────  │  ← 구분선
│                 │
│  ┌───────────┐  │
│  │           │  │
│  │  캐릭터   │  │  ← 선택된 에이전트의 캐릭터 이미지
│  │  이미지   │  │
│  │           │  │
│  └───────────┘  │
└─────────────────┘
```

에이전트를 전환하면 하단 캐릭터 이미지도 교체됨.

## 채팅 말풍선

```
비서 메시지 (왼쪽 정렬):
  ╭──────────────────────────────────────╮
  │ 오늘 미완료 미리알림이 3건 있어요.    │
  ╰──────────────────────────────────────╯

사용자 메시지 (오른쪽 정렬):
                      ╭──────────────────╮
                      │ 내일 일정은?     │
                      ╰──────────────────╯

스트리밍 중 (타이핑 커서):
  ╭──────────────────────────────────────╮
  │ 내일은 오후 2시에 ▌                  │
  ╰──────────────────────────────────────╯
```

## 캐릭터 이미지 상태

Phase 3에서는 더미 이미지, Phase 4에서 실제 상태 전환.

| 상태 | 트리거 |
|------|--------|
| `default` | 대기 중 |
| `thinking` | 응답 생성 중 |
| `happy` | 긍정적 응답 |
| `surprised` | 특정 키워드 반응 |

Live2D 적용 여부는 Phase 3 착수 시 결정.

## 스타일

| 항목 | 값 |
|------|----|
| 창 크기 | 720 × 580px (리사이즈 가능) |
| 사이드바 너비 | 160px |
| 사이드바 배경 | `#161618` |
| 메인 배경 | `#1c1c1e` |
| 비서 말풍선 | `#2c2c2e` 배경, `#ebebf5` 텍스트 |
| 사용자 말풍선 | `#2563eb` 배경, `#ffffff` 텍스트 |
| 폰트 | `-apple-system` |

## 입력바

팝업의 InputBar 컴포넌트 재사용. 앱 창 하단에 고정.
